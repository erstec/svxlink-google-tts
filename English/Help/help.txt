This is a S v x Link system. It is a multi purpose voice services system for ham radio use. S v x Link is controlled by DTMF commands consisting of one or more digits: followed buy a number sign.
Sending a single star,will make the system identify itself. 
The system is divided into several modules, each performing a specific task.
You activate a module by sending its module ID, followed buy a number sign.
Most modules will play their help message if you send a 0 while its activated.
